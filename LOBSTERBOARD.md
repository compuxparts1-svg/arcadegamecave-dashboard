# 🦞 LobsterBoard Dashboard

Professional, drag-and-drop dashboard builder for monitoring ArcadeGameCave business metrics.

## Quick Start

```bash
# Start the dashboard server
./lobsterboard-start.sh

# Or manually:
PORT=8080 node node_modules/lobsterboard/server.cjs
```

Then open: **http://localhost:8080**

## Edit Mode

1. Open the dashboard
2. Press **Ctrl+E** (or look for Edit button)
3. Drag widgets around, resize, configure
4. Click **💾 Save** when done

## Available Widgets

### 💰 Custom: Daily Profits (NEW)
Shows real-time profit data from all sales channels:
- Total profit across all platforms
- eBay breakdown (item sales, shipping, fees, profit)

- Calendar (Google, Apple, Outlook iCal feeds)
- Finance (stocks, crypto)
- Productivity (todos, notes, pomodoro timer)
- Media (RSS feeds, now playing, quotes)
- Smart home (camera feeds, power usage, temperature)
- OpenClaw integration (cron jobs, sessions, activity logs)
- And many more...

## Custom Profit Widget

**Location:** `/community-widgets/daily-profits.js`

**Features:**
- Reads `.ebay-daily-profit.json` (generated by cron job)
- Displays:
  - Total daily profit (30% margin)
  - Orders count
  - Item sales breakdown
  - Shipping costs
  - eBay fees
  - Net amount received
  - Timestamp of last update

**Multi-Platform Ready:**
- eBay: ✅ Live
- Mercari: ⏳ Pending (waiting for API credentials)
- Other platforms: 🔜 Ready to add

## Widget Properties

When in edit mode, click a widget to configure:

**Daily Profits Widget Settings:**
- `title` - Widget title (default: "Daily Profits (All Platforms)")
- `refreshInterval` - How often to check for new data (seconds, default: 30)

## Data Integration

The profit widget automatically reads from:
- **File:** `.ebay-daily-profit.json`
- **Generated by:** `ebay-daily-profit.js` (cron job at 8 AM MT)
- **Format:** JSON with `date`, `orders`, `itemSales`, `shipping`, `fees`, `youReceived`, `profit`, `byProduct`

## Custom Pages

Add full pages beyond the dashboard widget grid:

1. Create a folder in `pages/` (e.g., `pages/my-page/`)
2. Add files:
   - `page.json` - metadata (title, icon, order)
   - `index.html` - page UI
   - `api.cjs` (optional) - server-side API routes

Pages auto-discover on startup.

## API Endpoints

**Dashboard Data:**
- `GET /config` - Load dashboard layout
- `POST /config` - Save dashboard layout

**Profits Data:**
- `GET /.ebay-daily-profit.json` - Daily profit report (read by widget)

**LobsterBoard API:**
- `GET /api/stats/stream` - Live system stats (Server-Sent Events)
- `GET /api/templates` - List dashboard templates
- `POST /api/templates/export` - Export current dashboard
- `POST /api/templates/import` - Import a template

## Systemd Service (Optional)

Run LobsterBoard as a service on Linux:

```bash
sudo cat > /etc/systemd/system/lobsterboard.service << 'EOF'
[Unit]
Description=LobsterBoard Dashboard
After=network.target

[Service]
Type=simple
User=jason
WorkingDirectory=/home/jason/.openclaw/workspace
ExecStart=/usr/bin/node node_modules/lobsterboard/server.cjs
Environment=PORT=8080 HOST=0.0.0.0
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
EOF

sudo systemctl daemon-reload
sudo systemctl enable lobsterboard
sudo systemctl start lobsterboard
```

## Troubleshooting

**Widget not showing data?**
- Make sure `ebay-daily-profit.js` has run once (generates `.ebay-daily-profit.json`)
- Refresh the browser (Ctrl+R or Cmd+R)
- Check browser console for errors (F12)

**Port 8080 already in use?**
```bash
PORT=3000 node node_modules/lobsterboard/server.cjs
# Then visit http://localhost:3000
```

**Can't find custom widgets?**
- Reload the page
- Make sure `community-widgets/daily-profits.js` exists
- Check browser console for JavaScript errors
